# Screenshots

Visual evidence from lab exercises.

## Guidelines

- **Sanitize:** Remove or redact any real passwords, hashes, or sensitive data
- **Annotate:** Add boxes/arrows to highlight important areas
- **Name clearly:** Use descriptive filenames

## Naming Convention

`YYYY-MM-DD_description.png`

Example: `2025-11-21_nmap-dc-scan.png`

## Tools for Annotation

- Flameshot (Linux)
- Greenshot (Windows)
- ShareX (Windows)
