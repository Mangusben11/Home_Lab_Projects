# Writeups

Detailed technical writeups of attacks and techniques practiced in the home lab.

## Format

Each writeup should include:
- **Objective:** What you're trying to accomplish
- **Environment:** Target systems and configuration
- **Steps:** Detailed walkthrough with commands
- **Results:** What happened, what you found
- **Defense:** How to detect/prevent this attack
- **MITRE ATT&CK:** Relevant technique IDs

## Naming Convention

`YYYY-MM-DD_technique-name.md`

Example: `2025-11-21_kerberoasting.md`
